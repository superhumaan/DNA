# Memory import

```bash
npx dna memory import <file> --merge
npx dna memory import <file> --on-conflict newest
```

Conflict strategies: `newest` (default), `keep-local`, `keep-remote`.

File: $ARGUMENTS

Confirm merge result. Never overwrite without user consent.
