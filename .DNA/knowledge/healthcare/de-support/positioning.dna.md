Pair with `healthcare/overview-de`. Auto-installed when you run `dna marketplace install healthcare/overview-de` or `dna marketplace install healthcare/germany`.

- `gematik.dna.md` — Gematik TI Gateway
- `epa.dna.md` — ePA (Electronic Patient Record)
- `diga.dna.md` — DiGA (Digital Health Apps)
- `compliance.dna.md` — Compliance Actions

dna marketplace install healthcare/overview-de
