# Healthcare Support — United States

HIPAA operational controls, CMS interoperability APIs, payer FHIR (Da Vinci), and BAA-covered vendor management.

Pair with `healthcare/overview-us`. Auto-installed when you run `dna marketplace install healthcare/overview-us` or `dna marketplace install healthcare/united states`.

## Topics in this pack
- `systems-integrations.dna.md` — Healthcare Systems & Integration
- `hipaa-ops.dna.md` — HIPAA Operations
- `cms-apis.dna.md` — CMS Patient & Provider Access APIs
- `payer-fhir.dna.md` — Payer & RCM Integration
- `compliance.dna.md` — Compliance Actions

## DNA commands
```bash
dna marketplace install healthcare/overview-us
dna plan compliance --frameworks <see compliance topic>
dna context healthcare
```
