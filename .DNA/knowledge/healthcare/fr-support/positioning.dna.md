# Healthcare Support — France

HDS-certified hosting, DMP/Mon Espace Santé, ANS interoperability, and Pro Santé Connect for clinician identity.

Pair with `healthcare/overview-fr`. Auto-installed when you run `dna marketplace install healthcare/overview-fr` or `dna marketplace install healthcare/france`.

## Topics in this pack
- `systems-integrations.dna.md` — Healthcare Systems & Integration
- `hds.dna.md` — HDS Hosting (Hébergeur de Données de Santé)
- `dmp.dna.md` — DMP & Mon Espace Santé
- `ans.dna.md` — ANS Interoperability (CI-SIS FHIR)
- `compliance.dna.md` — Compliance Actions

## DNA commands
```bash
dna marketplace install healthcare/overview-fr
dna plan compliance --frameworks <see compliance topic>
dna context healthcare
```
