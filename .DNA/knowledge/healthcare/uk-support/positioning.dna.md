# Healthcare Support — United Kingdom

NHS DSPT assurance, GP Connect, CIS2 clinician auth, and UK Core FHIR conformance.

Pair with `healthcare/overview-uk`. Auto-installed when you run `dna marketplace install healthcare/overview-uk` or `dna marketplace install healthcare/united kingdom`.

## Topics in this pack
- `systems-integrations.dna.md` — Healthcare Systems & Integration
- `dspt.dna.md` — NHS DSPT & DTAC
- `gp-connect.dna.md` — GP Connect & Spine
- `uk-core.dna.md` — UK Core FHIR
- `compliance.dna.md` — Compliance Actions

## DNA commands
```bash
dna marketplace install healthcare/overview-uk
dna plan compliance --frameworks <see compliance topic>
dna context healthcare
```
