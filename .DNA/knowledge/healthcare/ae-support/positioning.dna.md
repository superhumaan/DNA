# Healthcare Support — United Arab Emirates

UAE PDPL, Malaffi (Abu Dhabi) and NABIDH (Dubai) HIE programmes.

Pair with `healthcare/overview-ae`. Auto-installed when you run `dna marketplace install healthcare/overview-ae` or `dna marketplace install healthcare/united arab emirates`.

## Topics in this pack
- `systems-integrations.dna.md` — Healthcare Systems & Integration
- `systems.dna.md` — National Health Systems (summary)
- `integration.dna.md` — Integration Patterns
- `compliance.dna.md` — Compliance Actions

## DNA commands
```bash
dna marketplace install healthcare/overview-ae
dna plan compliance --frameworks <see compliance topic>
dna context healthcare
```
