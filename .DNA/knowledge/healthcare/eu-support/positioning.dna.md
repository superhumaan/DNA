# Healthcare Support — European Union

Cross-border EHDS readiness, EU MDR SaMD, GDPR Art. 9 health data, and national IG overlays.

Pair with `healthcare/overview-eu`. Auto-installed when you run `dna marketplace install healthcare/overview-eu` or `dna marketplace install healthcare/european union`.

## Topics in this pack
- `systems-integrations.dna.md` — Healthcare Systems & Integration
- `gdpr-health.dna.md` — GDPR Health Data (Art. 9)
- `mdr.dna.md` — EU MDR & SaMD
- `ehds.dna.md` — European Health Data Space
- `compliance.dna.md` — Compliance Actions

## DNA commands
```bash
dna marketplace install healthcare/overview-eu
dna plan compliance --frameworks <see compliance topic>
dna context healthcare
```
