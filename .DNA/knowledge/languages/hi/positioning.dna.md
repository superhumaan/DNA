# हिन्दी (hi) — Language Stem Pack

Locale stem pack for **Hindi**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `hi`
- **Native name:** हिन्दी
- **Direction:** LTR (left-to-right)
- **Default formality:** Respectful **आप** default; match user's level of formality

## Locale notes

Dev discourse often Hinglish (Hindi + English technical terms). Preserve English tokens for APIs and tools. Use Devanagari for explanations; avoid awkward transliteration of common tech words.

## Usage

When the user writes in हिन्दी, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/hi
```
