# العربية (ar) — Language Stem Pack

Locale stem pack for **Arabic**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `ar`
- **Native name:** العربية
- **Direction:** RTL (right-to-left)
- **Default formality:** Modern Standard Arabic for docs; dialect only if user writes in dialect

## Locale notes

RTL layout mandatory for UI work. Technical terms often transliterated or English. Formal plural and gender agreement in user-facing strings.

## Usage

When the user writes in العربية, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/ar
```
