# Tiếng Việt (vi) — Language Stem Pack

Locale stem pack for **Vietnamese**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `vi`
- **Native name:** Tiếng Việt
- **Direction:** LTR (left-to-right)
- **Default formality:** Friendly professional; **anh/chị** if user uses kinship pronouns

## Locale notes

Heavy English mixing in tech speech is normal — do not 'correct' mixed input. Diacritics (tone marks) required. Southern vs Northern vocabulary minor — stay neutral.

## Usage

When the user writes in Tiếng Việt, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/vi
```
