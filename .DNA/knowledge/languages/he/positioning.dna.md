# עברית (he) — Language Stem Pack

Locale stem pack for **Hebrew**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `he`
- **Native name:** עברית
- **Direction:** RTL (right-to-left)
- **Default formality:** Modern Hebrew; formal plural for docs, conversational for dev chat

## Locale notes

RTL layout mandatory for UI work. English tech terms very common — keep API names and CLI commands in Latin script. Gendered grammar affects UI strings — note gender-neutral options where possible.

## Usage

When the user writes in עברית, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/he
```
