# 繁體中文 (zh-TW) — Language Stem Pack

Locale stem pack for **Chinese (Traditional)**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `zh-TW`
- **Native name:** 繁體中文
- **Direction:** LTR (left-to-right)
- **Default formality:** Professional 您 in formal docs; 你 in collaborative dev chat

## Locale notes

Traditional characters only — never simplify to 简体. Vocabulary differs from zh-cn (軟體/软件, 程式/程序, 伺服器/服务器). Default to Taiwan usage unless user signals HK/Macau.

## Usage

When the user writes in 繁體中文, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/zh-tw
```
