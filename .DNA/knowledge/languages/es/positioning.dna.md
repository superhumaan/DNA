# Español (es) — Language Stem Pack

Locale stem pack for **Spanish**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `es`
- **Native name:** Español
- **Direction:** LTR (left-to-right)
- **Default formality:** Use **tú** for dev tooling chat unless user uses **usted**; formal for compliance docs

## Locale notes

Watch European vs Latin American vocabulary (ordenador/computadora, archivo/fichero). Default to neutral international Spanish unless user signals region.

## Usage

When the user writes in Español, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/es
```
