# 简体中文 (zh-CN) — Language Stem Pack

Locale stem pack for **Chinese (Simplified)**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `zh-CN`
- **Native name:** 简体中文
- **Direction:** LTR (left-to-right)
- **Default formality:** Professional 您 in formal docs; 你 in collaborative dev chat

## Locale notes

Simplified characters only unless user requests Traditional (zh-TW). Technical terms often stay English or use established 中文 equivalents.

## Usage

When the user writes in 简体中文, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/zh-cn
```
