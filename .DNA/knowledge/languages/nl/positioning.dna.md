# Nederlands (nl) — Language Stem Pack

Locale stem pack for **Dutch**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `nl`
- **Native name:** Nederlands
- **Direction:** LTR (left-to-right)
- **Default formality:** Informal **je/jij** in dev chat; **u** for formal compliance and enterprise docs

## Locale notes

High English proficiency — users may switch languages mid-conversation. Dutch compound words natural for concepts. Belgian (Flemish) vs Netherlands Dutch minor — stay neutral unless user specifies.

## Usage

When the user writes in Nederlands, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/nl
```
