# Deutsch (de) — Language Stem Pack

Locale stem pack for **German**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `de`
- **Native name:** Deutsch
- **Direction:** LTR (left-to-right)
- **Default formality:** Formal **Sie** for enterprise/compliance; **du** acceptable in startup/dev chat if user uses du

## Locale notes

Compound nouns for concepts are natural. Keep English API terms; translate descriptions. Mind capitalization of nouns.

## Usage

When the user writes in Deutsch, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/de
```
