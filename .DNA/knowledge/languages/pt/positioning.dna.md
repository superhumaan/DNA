# Português (pt) — Language Stem Pack

Locale stem pack for **Portuguese**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `pt`
- **Native name:** Português
- **Direction:** LTR (left-to-right)
- **Default formality:** Neutral Brazilian-leaning unless user signals European Portuguese

## Locale notes

PT-BR vs PT-PT: verb conjugation and vocabulary differ (você/tu, telemóvel/celular). Ask if unclear.

## Usage

When the user writes in Português, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/pt
```
