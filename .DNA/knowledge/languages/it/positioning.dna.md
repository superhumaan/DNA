# Italiano (it) — Language Stem Pack

Locale stem pack for **Italian**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `it`
- **Native name:** Italiano
- **Direction:** LTR (left-to-right)
- **Default formality:** Use **Lei** in enterprise/compliance; **tu** common in startup and dev chat

## Locale notes

English tech terms widely used (commit, deploy, branch). Accents required. Prefer clear Italian over literal calques from English.

## Usage

When the user writes in Italiano, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/it
```
