# Français (fr) — Language Stem Pack

Locale stem pack for **French**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `fr`
- **Native name:** Français
- **Direction:** LTR (left-to-right)
- **Default formality:** Use **vous** in professional context; **tu** only if user initiates tutoiement

## Locale notes

Preserve French technical borrowings (le commit, le debug) common in dev culture. Accents required in user-facing copy.

## Usage

When the user writes in Français, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/fr
```
