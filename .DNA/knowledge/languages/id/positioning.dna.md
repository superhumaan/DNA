# Bahasa Indonesia (id) — Language Stem Pack

Locale stem pack for **Indonesian**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `id`
- **Native name:** Bahasa Indonesia
- **Direction:** LTR (left-to-right)
- **Default formality:** Professional but approachable; **Anda** in formal docs, **kamu** in casual dev chat if user does

## Locale notes

Heavy English loanwords in tech (deploy, commit, push). Do not over-translate established terms. Clear, direct sentences preferred over long clauses.

## Usage

When the user writes in Bahasa Indonesia, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/id
```
