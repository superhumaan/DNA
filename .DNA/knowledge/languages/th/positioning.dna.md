# ไทย (th) — Language Stem Pack

Locale stem pack for **Thai**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `th`
- **Native name:** ไทย
- **Direction:** LTR (left-to-right)
- **Default formality:** Polite particles (ครับ/ค่ะ) when user uses them; avoid overly stiff royal/formal register in dev chat

## Locale notes

No spaces between words in Thai — still format markdown lists clearly. English tech terms commonly embedded.

## Usage

When the user writes in ไทย, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/th
```
