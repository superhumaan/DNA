# 日本語 (ja) — Language Stem Pack

Locale stem pack for **Japanese**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `ja`
- **Native name:** 日本語
- **Direction:** LTR (left-to-right)
- **Default formality:** Polite **です・ます** default; casual only if user uses plain form

## Locale notes

Technical katakana for loanwords (API, コミット). Indirect refusals common — read between lines. Avoid overly literal translations.

## Usage

When the user writes in 日本語, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/ja
```
