# 한국어 (ko) — Language Stem Pack

Locale stem pack for **Korean**. Pair with `languages/stem-bridge`.

- **ISO 639-1:** `ko`
- **Native name:** 한국어
- **Direction:** LTR (left-to-right)
- **Default formality:** Honorific **합니다** default; match user's speech level

## Locale notes

Hangul for explanations; Latin for code. Age/hierarchy affects formality — follow user's lead.

## Usage

When the user writes in 한국어, load this pack via `dna context multilingual` or install:

```bash
dna marketplace install languages/ko
```
