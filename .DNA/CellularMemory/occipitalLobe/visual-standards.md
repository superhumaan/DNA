# Visual Standards

## DNA Lab

- Humaan-aligned neutral shell with clear status hierarchy
- Minimum 16px base text, visible focus states, and keyboard-operable controls
- Dense operational tables remain scannable without hiding severity or timestamps

## DNA-Web

- Background/surface/border/text/accent come from `dna-*` Tailwind tokens
- Display headings use Syne; supporting copy uses DM Sans
- Rounded report panels use subtle borders and restrained accent fills
- Health states pair semantic labels with accessible contrast and explanatory copy
- Dates include timezone/recency context; metrics include units and source links
