# Architecture Overview

_Generated by DNA by Humaan. Review and refine as your project evolves._

## System summary

**dna-by-humaan** is a TypeScript monorepo that ships the DNA CLI (`@superhumaan/dna-by-humaan`) and supporting packages for project intelligence, runtime observation, knowledge packs, compliance planning, and AI workbench integration.

## Monorepo layout

| Area | Purpose |
|------|---------|
| `packages/dna-cli` | Published CLI binary (`dna`) |
| `packages/dna-core` | Analysis, generators, quality, marketplace, IVF |
| `packages/dna-config` | Config schemas and validation |
| `packages/dna-github` | Git and GitHub automation |
| `packages/dna-immune` | Runtime error classification |
| `packages/dna-runtime` | Runtime observer middleware |
| `packages/dna-ai` | AI provider orchestration |
| `packages/dna-feedback` | Upstream feedback reporting |
| `packages/dna-templates` | Scaffolding templates |
| `apps/examples` | Example integrations (Express, Vite) |
| `apps/marketplace` | Local marketplace dev server |
| `.DNA/` | Project intelligence (behaviour, knowledge, memory) |
| `DNA/Impressions/` | Living documentation |

## Core flows

1. **Init / doctor** — scaffold `.DNA/`, CI, hooks, workbench, stems
2. **Analyze / scan** — structure, auth, integrations, gaps
3. **Quality gate** — lint, typecheck, tests, SAST patterns
4. **Ship** — docker build, GitHub push, npm publish via Actions
