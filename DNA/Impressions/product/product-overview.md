# Product Overview

## What DNA is

DNA by Humaan is an open-source engineering co-pilot and project-intelligence
system. One CLI installs durable architecture knowledge, behavior rules,
CellularMemory, runtime observation, DNA Lab, quality gates, feature delivery
workflows, and a large offline-first knowledge marketplace.

## Users

- Product engineering teams using Cursor or Claude Code
- Maintainers standardizing quality and delivery across a portfolio
- Operators investigating runtime issues through DNA Lab
- npm consumers embedding runtime/Lab adapters

## Product characteristics

- Plain-language workflows backed by real repository context
- Zero required production dependencies in the public CLI
- Poll-based, responsive operational UI without a socket requirement
- Fail-closed security and deployment boundaries
- Transparent health evidence across GitHub, npm, and dna.humaan.app

**Stage:** scaling · **Compliance mode:** project-configurable
