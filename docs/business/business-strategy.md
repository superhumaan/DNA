# Business strategy

DNA by Humaan is an **open-source project intelligence platform** for TypeScript teams. The business case is simple: AI tools are only as good as the context they receive — and most repos ship code without a durable memory of *why* the system works the way it does.

---

## Vision

**Every TypeScript team** should have a project brain that:

- Gives AI tools structured, project-specific context
- Observes production and classifies issues with system memory
- Ships compliance and platform patterns proportionate to org size
- Automates GitHub issues and repair workflows — without auto-merging or touching secrets

Git remembers your code. **DNA remembers your system.**

---

## Strategic pillars

1. **Open by default** — MIT license, npm distribution, GitHub-first development
2. **Proportionate compliance** — startup baselines through enterprise GRC, not one-size-fits-all
3. **Production-learned patterns** — platform catalog distilled from AIStudio, ColorParty, Humaan Ops, and Soli
4. **One package, two modes** — CLI for intelligence; `/runtime` subpath for production observation
5. **Marketplace scale** — 897 knowledge packs with offline bundled fallback

---

## Positioning

| Competitor mental model | DNA difference |
|-------------------------|----------------|
| Documentation generators | DNA is not a doc generator — it is machine-readable intelligence + human Impressions |
| Error trackers (Sentry) | DNA explains *why* with Behaviour, CellularMemory, and opens contextual PRs |
| Generic AI rules | Behaviour + neuralNetwork route intent → only relevant knowledge |
| Compliance consultants | Tiered plans + installable knowledge packs AI can execute |
| Per-repo AI memory (Cursor) | **Portfolio install** — one `.DNA/` at the squad folder; every product + interconnections |

---

## Portfolio positioning

**One squad, many products, one install.** Run `dna doctor` on the parent folder — not per repo. One command scaffolds **59 stem packs**, **Cursor + Claude rules and skills**, **automatic detection** across every app, **CellularMemory**, **runtime observer**, **CI/hooks**, **feature factory**, and **897 knowledge packs**. Cross-product system maps, shared memory, same feature template every time. Enterprise platform-team capability without Backstage + Sentry + premium AI memory bills — the install teams keep like `tsconfig.json`.

## What success looks like

| Metric | Target behaviour |
|--------|------------------|
| **Adoption** | `dna init` in new and brownfield TypeScript repos |
| **Daily use** | `dna context cursor` before AI sessions; `dna doctor` in CI |
| **Runtime** | Express/Fastify/NestJS/Next.js APIs protected in production |
| **Marketplace** | Teams install stack + compliance packs; `dna update` keeps current |
| **Platform features** | `dna plan feature` used for admin, SSO, RBAC, deploy patterns |
