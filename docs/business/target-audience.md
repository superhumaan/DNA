# Target audience

DNA serves TypeScript teams building web APIs, full-stack apps, and regulated products — from solo founders to enterprise engineering orgs.

---

## Primary users

| Persona | What they need from DNA |
|---------|-------------------------|
| **Squad / platform lead** | [Portfolio install](../product/portfolio-install.md) — one brain for every related product |
| **Individual contributor** | `dna context cursor` — AI that knows the project's rules and stack |
| **Tech lead** | Brownfield IVF plans, architecture Impressions, Behaviour governance |
| **Platform / DevOps** | Runtime observer, CI hooks, Docker build gates, GitHub automation |
| **Security / compliance** | Tiered GDPR/HIPAA/ISO/SOC 2 plans + installable knowledge packs |
| **Product / PM** | `DNA/Impressions/product/` — human docs separate from machine intelligence |

---

## Organisation tiers

DNA infers compliance posture from project stage and org size:

| Tier | Size | DNA delivers |
|------|------|--------------|
| **Startup** | 1–25 | Baselines — TLS, secrets, auth, privacy notice |
| **SME** | 25–250 | Policies, access reviews, DPIA templates |
| **Corporate** | 250–5K | GRC programme, pen tests, certification prep |
| **Enterprise** | 5K+ / regulated | Multi-framework mapping, continuous monitoring |

See [Compliance tiers](../product/compliance-tiers.md).

---

## Ideal project profiles

- **Portfolio / multi-product squad** — one `dna doctor` on the parent folder ([guide](../product/portfolio-install.md))
- **Greenfield** — `dna doctor` scaffolds `.DNA/` + `DNA/Impressions/` + workbench
- **Brownfield** — `dna analyze` + `dna plan ivf` for legacy modernisation
- **API servers** — runtime observer for Express, Fastify, NestJS, Next.js
- **Healthcare / fintech** — HIPAA, FHIR, PCI packs in marketplace
- **Multi-tenant SaaS** — platform catalog patterns from Soli and AIStudio

---

## Not the target (today)

- Non-TypeScript primary stacks (limited pack coverage)
- Teams wanting only static documentation sites
- Orgs requiring hosted SaaS-only (DNA is self-hosted CLI + runtime)
